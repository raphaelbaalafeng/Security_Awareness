### Cyber crimes ### 

# Vishing
-a technique of social engineering wherby cybercriminals use phone calls, voice messages to trick people to reveal sensitive information.
# Phishing
-an attack whereby criminals send impersonate legitimate looking emails to trick one into reveling confidential data. 
# Denial of Service Attack
-an attack that floods a network or service by flooding it with traffic or requests.
# Malware
-malicious software or code designed for disruption and to gain unauthorized access to computer systems etc.

# Spyware
-a malicious software that gathers information by monitoring key presses on a user's keyboard, the information is then sent back to the sender.
